Complete Guide to Markdown shortcuts for typora

Created2025-01-21|Updated2025-01-21|Tools
|Word Count:858|Reading Time:3mins|Comments:

文字样式部分

标题

下划线

P.s.下划线(Underline)，因此 +u。

删除线

字体加粗

P.s.粗体的(Bold)，因此 **+b**。

字体倾斜

P.s.斜体的(Italic)，因此 **+i**。

内容部分

引用

P.s.引用(Quote)，因此 **+q**。

插入链接

P.s.超链接(Hyperlink)，因此 **+k**。

插入图片

P.s.超链接(Image)，因此 **+i**。

文档内部跳转

P.s.home：起始 end：终止

选择英文单词/中文

P.s.向下填充(Down)，因此 **+d**。

按行选中文本

P.s.排成一行(Line)，因此 **+l**。

快速搜索文本内容

P.s.找寻(Find)，因此 **+f**。

快速替换文本

P.s.替换(Replace)，但R键位被占用，转而使用 **+h**。

快速生成表格

P.s.表格(Table)，因此 **+t**。

快速打开最近浏览笔记

P.s.呈现(Present)，因此 **+p**。

快速生成目录

P.s.目录表(Table Of Contents )，因此 **[toc]**。

着重关键字

快速创建代码块(自定义语言类型)

数学表达式(支持Latex语法)

  • 行内公式

    例如不定积分公式：$\int f(x)dx = F(x) + c (c\in {\forall} constants)$

  • 行外公式

    P.s.数学(Mathematics)，因此 **+m**。

    例如单调收敛定理：

    $$
    \forall x \in E , 0 \leq f_n(x) \leq f_{n+1}(x),n\in {N_+}.且\lim\limits_{n\to\infty}f_n(x)=f(x).
    那么，\lim\limits_{n\to\infty}\int_Ef(x)dx=\int_Ef(x)dx.
    $$

工具自身部分

新建文件

P.s.新建(New)，因此 **+n**。

打开已有.md文件

P.s.打开(Open)，因此 **+o**。

保存.md文件

P.s.保存(Save)，因此 **+s**。

关闭.md文件

P.s.窗口(Window)，因此 **+w**。
